What is the diffrence between session storage and local storage?

The key contrast between session storage and local storage is that local storage data typically does not erase from the storage, even when the browser is resumed, but session storage data is often lost when the page is closed. The local storage attribute is used to access a page's local storage area without a time limit, and it will be available whenever you use that page. The basic difference among local and session storage is that local storage has no expiration date, however session storage data is deleted when the browser tab is closed. We cant get the data after browser closed as session has out, thus the name -session. Intrestingly say, the Javascript DOM allows access to both storages.
